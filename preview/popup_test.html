これは離脱ポップアップ(popup.js)動作確認用の偽LPです

テスト手順(このページの設定: delay5000 + back / oncePer:'load')
① 何も触らず5秒待つ → ポップアップが出る(トリガー=delay)。×で閉じる
② リロード → ページを一度タップかスクロールしてから「戻る」 → ポップアップが出る(トリガー=back)
③ ②の後もう一度「戻る」 → 引き止めずに普通にページを離れる(無限トラップしない)
④ ポップアップのCTAボタン → チャットが起動する
⑤ リロード → 先にチャットを開いたまま5秒待つ → ポップアップは出ない(チャット優先)。チャットを×で閉じると、その後の再判定(10秒おき)で出る
⑥ リロード → 下のフォーム欄にカーソルを入れたまま5秒待つ → 出ない。欄の外をタップすると再判定で出る
左下の黒い枠に hs_chat_popup_* イベントが流れることも確認(計測の確認)

URLパラメータで切替できるテストモード
?img … 画像モード(画像+重ねCTAボタン。画像は押せずボタンだけ反応)
?img&clickable … ada-cloud式(重ねボタンなし・画像全体がCTA)
?backonly … ブラウザバックのみ(delay無効)
?chatclose … チャットを開いて×で閉じた瞬間に出るテスト
?closeconfirm … チャットの×で「閉じますか？」確認ダイアログ(白部分+クリエイティブ)。?closeconfirm&noimg で画像なし版
